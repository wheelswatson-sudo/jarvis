# What Your Assistant Can Build

Examples of things you can ask your assistant to build. All of these are doable in Claude Code.

## System Automation

- Morning briefing scripts (email summary, calendar, weather, news)
- Boot scripts that start your dev environment in one command
- File organization routines (auto-sort Downloads, rename screenshots)
- System health monitors (disk, battery, network, CPU)
- Backup scripts (to external drive, cloud, git)
- Focus mode toggles (disable notifications, close distracting apps)

## Personal CRM & Contacts

- Track people you meet (name, context, follow-up dates)
- Birthday/anniversary reminders with AI-generated messages
- Conversation memory (remember what each person told you)
- Follow-up nudges ("you haven't talked to X in 3 weeks")
- Relationship strength scoring

## Knowledge & Learning

- Second brain (atomic notes, linking, tag system)
- Research assistant (web search, summarize, file in categories)
- Reading list manager (highlights, summaries, re-surfacing)
- Meeting note-taker (structured summaries, action items)
- Learning trackers (courses, books, skills with progress)

## Communication

- Email drafting (reads your style from past emails, drafts new ones)
- Calendar management (schedule, reschedule, decline politely)
- iMessage/SMS automation (via Mac Messages integration)
- Slack summaries (catch up on missed channels)
- LinkedIn content engine (drafting, scoring, scheduling)

## Development & Work

- Git workflow automation (branch naming, commit messages, PR descriptions)
- Code review bot (pre-commit checks, style enforcement)
- Project scaffolding (new repo with your preferred structure)
- Test running automation
- Deployment scripts (build, test, ship)
- Documentation generators

## Business & Revenue

- Competitive research tools
- Lead enrichment (take a name, pull public info)
- Outreach sequences (cold DM templates, reply tracking)
- Content scheduling (blog, social, newsletter)
- Analytics dashboards (custom metrics for your business)
- Invoice/expense tracking

## Health & Habits

- Habit trackers (streaks, reminders, accountability)
- Workout loggers
- Food/nutrition trackers
- Sleep pattern analysis
- Mood/energy journaling with trend analysis
- Goal progress dashboards

## Household

- Grocery list management (shared with partner, auto-generated from meal plans)
- Meal planning (based on dietary preferences, what's in fridge)
- Bill tracking and payment reminders
- Home inventory (what's where, when bought, warranty dates)
- Package tracking (consolidates all shipments)

## Integrations Available

Your assistant can connect to (via MCP servers):

- Google Calendar, Gmail, Drive, Docs, Sheets
- iMessage, Contacts, Notes (Mac)
- Spotify, Apple Music
- GitHub, Linear, Notion, Slack
- Any API with an access token

---

## How to Ask

Bad prompts:
- "Can you help me with email?"
- "I need to be more organized."

Good prompts:
- "Build me a script that runs every morning at 7am, reads my inbox, and texts me a 3-sentence summary of anything that actually matters."
- "I keep forgetting to follow up with people I meet. Build me a CRM in Apple Notes that reminds me to check in with someone every 2 weeks if I haven't talked to them."

**Be specific about the outcome. Let your assistant figure out the how.**
