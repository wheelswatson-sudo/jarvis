# Quick Start — Your First 5 Minutes

## 1. Install (2 minutes)

```bash
bash install.sh
```

Answer the prompts:
- Name your assistant (e.g., "Jarvis", "Jeeves", "Friday")
- Your name
- ElevenLabs API key (get one free at [elevenlabs.io](https://elevenlabs.io/app/settings/api-keys))

## 2. Open Claude Code (30 seconds)

```bash
# Open a new terminal, then:
claude
```

Your assistant will introduce itself and ask three questions about what you're working on.

**Answer them honestly.** It uses your answers to propose something you can build together in the next 15 minutes.

## 3. Build Your First Win (15 minutes)

Your assistant will suggest ONE specific thing based on your answers. Examples:

- **"Email takes too long"** → It builds a morning briefing script
- **"I forget to follow up with people"** → It scaffolds a lightweight CRM
- **"My dev environment is messy"** → It writes a single-command boot script
- **"I want to track X"** → It builds a tracker

**Let it do the work.** Watch what it builds. This is the moment you understand what you have.

## 4. Turn On Voice (optional)

```bash
<your-assistant-name> on
```

Every response Claude gives is now spoken aloud. Natural, human voice.

Toggle anytime:
- `<name> on` — voice on
- `<name> off` — silent
- `<name> auto` — smart filtering (only important stuff)

## 5. Start Talking To It

```bash
<your-assistant-name>-listen
```

Press Enter, speak your question, press Enter. It transcribes and responds.

For continuous conversation:

```bash
<your-assistant-name>-converse --loop
```

Speak, get spoken response, speak again. Say "goodbye" to exit.

---

## What to Do Next

Once the first win is done, just **ask it for things**. It remembers what you worked on. It can build more.

Good prompts to try:

- "What could I automate about my morning routine?"
- "I have [problem]. What's the fastest way to solve it?"
- "Build me a [tool] that does [thing]"
- "Connect my [service] so you can read from it"
- "Remind me about [thing] tomorrow at [time]"
